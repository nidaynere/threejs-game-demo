# Playable assignment
In this assignment you will need to create a 3D 'merge' game.
A few examples of these games are: 'Merge Dragons!' and 'EverMerge'.
We have also provided a concept which you can use for inspiration.

## Information
- The required images can be found in the 'Img' folder.
- The required models can be found in the 'models' folder.
- We have provided the following libraries which you can use: 'TweenJS, PixiJS, ThreeJS'
- Feel free to use different libraries and/or change the coding setup.

## Requirements
- Create a drag and drop system based on a grid.
- Create the cars using new materials and textures.
- Create an end-card. (concept image 3)

## How To Run
- install node
- run 'npm install http-server -g'
- run 'http-server'
- open your browser and navigate to 'localhost:8080/index.html'
- to update the game after making changes, save the changes and hard refresh your browser (CTRL + F5 or shift + refresh)
